Note: Ce projet n'a aucun lien officiel avec le jeu Shabadabada ou Sylvie Barc.


Singotron est né de la frustration de joueurs de Shabadabada (http://www.shabadabada.com/)
de ne pouvoir jouer faute d'avoir pensé à prendre le jeu.

J'ai donc décidé d'en faire une version web qui pourrait être utilisée depuis un téléphone mobile.

Par respect pour l'auteur, Sylvie Barc (http://www.sylviebarc.net/), je ne fournis pas de liste de mot avec le moteur.
C'est à vous de retranscrire votre copie du jeu.

II. Installation
-------------------------

Pour commencer, vous devez télécharger la dernière version stable sur :
http://github.com/nleclerc/singotron/downloads

Ensuite, vous devez décompresser l'archive et ajouter dans le même dossier des fichiers de mots,
à savoir words-fr.txt et words-en.txt. Vous pouvez prendre en exemple les fichiers words-fr.txt.dist
et words-en.txt.dist.

Une fois les mots ajoutés, il ne vous reste plus qu'a mettre les fichiers sur votre serveur web
préféré et y accéder avec votre téléphone ou navigateur favori.
